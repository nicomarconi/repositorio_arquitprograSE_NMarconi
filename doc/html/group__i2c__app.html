The i2c_driver_install function will install the driver.

Parameters
[in]i2c_numis the I2C instance we are using, I2C_NUM_0
[in]modethe I2C mode, which is master in this case.
[in]slv_rx_buf_lenthe size of the RX and TX buffers.
[in]slv_tx_buf_lenthe buffers are only needed if the ESP32 is being used in the slave mode. Since we are using it as a master, we will keep these buffers as 0.
[in]intr_alloc_flagsis to set the interrupt flag for the I2C. We will keep it as 0 too, since we are not using any interrupts
