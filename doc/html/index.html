General Description

Final Project for the Course Embedded Systems Programming and Architecture. It involves the sensing and visualization of temperature
and humidity values. The project uses the Espressif ESP32 WROOM module, the DHT11 humidity sensor, and a LCD screen.

Hardware setup

For running this firmware it is necesary, DHT11 sensor, LCD screen with
Expansor module PCF8574AT chip, and ESP32_WROOM module kit.
